### 1.3 类图说明：
本类图参考实验二图书管理系统用例，主要包含11个类，登陆、系统管理员、图书管理员、读者、馆藏书目、预定记录、借<br>
书记录、还书、公告、逾期记录、图书。其中系统管理员类，主要方法有系统维护、账号管理（增删改查）、权限管理、公<br>
告更新与删除； 图书管理员类，由维护图书用例产生，图书增删改查、图书预定管理、借书登记、还书登记、个人信息修<br>
改；读者类，由维护读者信息用例产生，被维护读者信息、图书归还、预定图书、查看图书、查询借阅情况、取消预定等。
